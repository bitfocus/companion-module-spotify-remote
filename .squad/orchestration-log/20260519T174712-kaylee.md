### Core Changes

3. **FadeVolume Helper:** Created helper function with dynamic `setIntervalAsync` to schedule volume steps
4. **Action Definition:** Registered FadeVolume action with Companion module

### Technical Details
- **Rate Limiting:** 500ms minimum interval between Spotify API calls
- **Cancellation:** AbortController ensures clean interruption of in-progress fades
- **Step Calculation:** `numSteps = Math.floor(duration / 500)` maximizes smooth transitions
- **Non-blocking:** Dynamic mode prevents API call pile-up by waiting for each async call to complete
